# GPA Calculator

A command-line GPA calculator written in C++.

## About
I built this project to practice C++ fundamentals while creating
something useful for my life as a college student. It calculates
your weighted GPA based on your course grades and credit hours.

## Features
- Input multiple courses with grades and credit hours
- Calculates weighted GPA on a 4.0 scale
- (Version 2) Letter grade support (A, B, C, D, F)

## How to Run

### Requirements
- g++ compiler (GCC)

### Compile
g++ gpa_calculator.cpp -o gpa_calculator

### Run
./gpa_calculator
